embeddabl.es

This is a simple API service to request dynamic images and text. These can be used to embed anywhere that supports it, or for websites and other means. If you have any requests for embeds, let me know at louis@machin.dev.

This service is currently free and public while it can be supported as such.

User variables

To make your embeds more dynamic, the use of variables that you can set at any time is really useful. There is currently no validation for users so please do not use any sensitive data, and keep your token private.

Creating a user

Request:
/user/create

Note: Any query parameters given will be used to initialize user variables object.

Response:
{
  "token": <token>
}
Example:
https://embeddabl.es/v1/user/create?alias=Louis

Setting variables

Request:
/user/set?token={token}

Any query parameters given, excluding token, will be set to the user variables object.

Response:
{
  "success": <boolean>
}
Example:
https://embeddabl.es/v1/user/set?token=eXaMPl3s&my_state=finished

Getting variables

Request:
/user/get?token={token}&key={variable}

The key is the name of the variable you are requesting.

Response:
{
  "success": <boolean>
  key: <value>
}
Example:
https://embeddabl.es/v1/user/get?token=eXaMPl3s&key=alias
{
  "alias": "Louis"
}

Deleting variables

Request:
/user/delete?token={token}&key={variable}

The key is the name of the variable you are deleting.

Response:
{
  "success": <boolean>
}
Example:
https://embeddabl.es/v1/user/set?token=eXaMPl3s&key=alias
